## flow reset skills

Reset shared agent skills to their defaults

![flow reset_skills](tapes/reset_skills.gif)


```
flow reset skills [flags]
```

### Examples

```
  flow reset skills
  flow reset skills --force
```

### Options

```
  -f, --force   Skip confirmation prompt
  -h, --help    help for skills
```
