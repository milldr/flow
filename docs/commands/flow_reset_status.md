## flow reset status

Reset the global status spec to its default

![flow reset_status](tapes/reset_status.gif)


```
flow reset status [flags]
```

### Examples

```
  flow reset status
  flow reset status --force
```

### Options

```
  -f, --force   Skip confirmation prompt
  -h, --help    help for status
```
