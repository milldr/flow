## flow sync

Fetch and rebase worktrees onto their base branches

```
flow sync <workspace> [flags]
```

### Examples

```
  flow sync calm-delta
```

### Options

```
  -h, --help   help for sync
```

### Options inherited from parent commands

```
  -v, --verbose   Enable verbose debug output
```
