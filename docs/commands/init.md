# flow init

Create a new empty workspace with a generated ID. Optionally provide a human-friendly name.

![flow init](tapes/init.gif)

## Usage

```
flow init [name]
```

## Examples

```bash
flow init              # ID only (e.g. calm-delta)
flow init vpc-ipv6     # ID + name
```
